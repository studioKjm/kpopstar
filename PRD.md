
# NewsFlow AI Console — Product Requirements Document (PRD v4, Final)

## 1. Overview
본 제품은 kpopstar.ai.kr의 실제 기사 운영 흐름을 기반으로, 기자 → 편집자 → 데스크의 제작·검수·발행 과정을 하나의 CMS에서 처리하도록 설계된 **AI 기반 뉴스 관리 콘솔**이다.  
연예·엔터테인먼트 기사에 특화된 문체, 민감도 요소, 팩트체크, 트렌드 구조를 기반으로 **실무 수준의 자동화 기능**을 제공한다.

또한 실제 기사 3건(화사·엔믹스·르세라핌) 분석을 통해 발견한 **오탈자, 중복서술, 일정 일관성, 팩트 정확도, 문체 특성**을 AI 기능 요구사항에 반영하였다.

---

## 2. Problem Definition

## 2.1 내부 콘솔(CMS)에 포함되어야 할 핵심 기능 요구사항

현재 공개된 기사 품질 자체에는 큰 문제가 없으며, 문체·사실 관계·오탈자 등은 안정적으로 유지되고 있다. 또한 기사 작성 및 관리를 위해서는 **내부 콘솔(CMS)**에는 다음과 같은 기능들이 포함되어야 효율적인 기사 제작·검수·발행 프로세스를 구축할 수 있다.

### ✔ AI 기반 편집·검수 기능
- 문단 구조 분석  
- 문체 통일  
- 중복 문장 감지  
- 사실·일정·명칭 자동 검증  
- 민감도 분석 및 대체 문구 제안  

### ✔ AI 기반 태그·카테고리 자동 추천 기능
- 인물명·그룹명·이벤트명 등 콘텐츠 기반 태그 생성  
- 태그 클릭 시 자동 필터링 기능  
- K-POP 특화 카테고리(스타뉴스·엔터테인먼트·스타트렌드 등) 자동 분류  

### ✔ 실시간 인기 및 트렌드 기사 반영 기능
- 조회수 기반 인기 기사 자동 추천  
- 트렌드 데이터 분석(검색량·SNS 언급 등)  
- 자동 편성 또는 후보 리스트 제공  

### ✔ 기사 작성–검수–발행의 단일 워크플로우
- 작성 → AI 검수 → 편집자 확인 → 발행  
- 기록·버전 관리 및 변경 내역 추적  

### ✔ 엔터테인먼트 도메인 특화 일정·사실 관계 자동 검증
- 컴백일·앨범 발매일·티저 일정·공연 일정 검증  
- 매체별 상이한 일정 존재 시 ‘검증 필요’ 플래그 표시  

### ✔ 반복 기사 패턴에 대한 템플릿 제공
- 컴백 기사  
- 인터뷰 기사  
- 행사 및 공연 기사  
- AI가 자동으로 구조를 추천하거나 초안을 생성  

---


### 2.2 실제 기사 분석에서 발견가능한 문제점
AI가 수행할 수 있는 검수 포인트는 다음과 같다:

- **문체 문제:**  
  연예 기사 특유의 “임을 밝혔다/전했다/답했다/기대를 모은다”와 같은 문체가 자연스럽지만, 리드·본문 간 표현 중복 사례가 존재.
- **중복 정보:**  
  리드 문장과 본문이 동일한 사실을 반복하는 경우가 확인됨.
- **마크업 오류:**  
  엔믹스 기사에서 빈 bullet(`*`) 발생 → 에디터 자동 감지 필요.
- **팩트체크:**  
  발매일·콘셉트명·월드투어 일정 등 대부분 정확하나, 매체별 발매일 상이(NMIXX 사례)처럼 불일치 가능성 존재.
- **민감도 요소:**  
  논란 기사 대비 필수적이므로 자동 감지 엔진 필요.

이 자료 기반으로 PRD의 AI 기능 요구사항은 현실적 과제 중심으로 설계됨.

### 3.1 MVP 과제 목표
- Base44 또는 Gemini 기반 UI + AI 기능 포함
- 최소 1개 이상의 AI 기능 작동
- Base44 URL 제출
- 10분 이내 데모 가능

### 3.2 비즈니스 목표
- 기사 작성 및 검수 효율 30–50% 향상
- 문체·팩트 오류 및 민감도 실수 최소화
- 자동 태깅을 통한 SEO 최적화

## 6.2 AI 기능

### 🔵 1) AI 문체 통일 (연예 뉴스 전용)
실제 기사에서 사용되는 주요 문체 패턴 기반:
- “~임을 밝혔다.”
- “~라고 전했다.”
- “~라고 답했다.”
- “~기대를 모으고 있다.”
- “관심이 모인다.”

AI는 문단별 문체 차이를 분석하여 자연스러운 보도체 스타일로 통일하고, 적용 여부는 기자가 선택한다.

---

### 🔵 2) 고도화된 AI 팩트체크 기능  
실제 기사 분석 결과를 기반으로 다음 영역을 검사:

#### a. 인물/소속/그룹명 정확도
- 멤버 표기(엔믹스·르세라핌 등)와 소속사 표기 정확 여부 검증  
- 최신 공식 라인업과 비교해 변경된 요소 존재 시 경고

#### b. 날짜·시기·스케줄 검증
- 공연 시작일·발매일·티저 공개일 등 일정 교차 검증  
- 매체별 상이한 발매일 존재 시 “불일치 가능성” 표시

#### c. 문단 간 논리적 일관성
- 같은 정보를 반복적으로 서술한 경우 하이라이트 표시  
- “리드 요약 + 본문 확장” 구조로 자동 수정 제안

#### d. 민감도 분석
- 논란성·비방·혐오 문구 감지  
- 2차 피해 발생 위험 문장 표시  
- 안전 대체 문구 제안

---

### 🔵 3) AI 자동 태그 생성 + 태그 검색 기능
본문 기반 자동 태그 예시:
- `#뉴진스 #하니 #컴백`
- `#엔믹스 #BlueValentine`
- `#르세라핌 #SPAGHETTI`

태그 클릭 → 해당 태그로 자동 필터링  
`/admin/articles?tag=뉴진스`

---

### 🔵 4) AI 중복 정보 탐지
실제 기사에서 발견된 문제 기반:
- 리드와 본문에서 동일 사실 반복  
- 동일 문체 반복(“강렬한 인상”, “눈길을 끈다” 등)  
- AI는 중복 문장 하이라이트 + 대체 문장 제안

---

### 🔵 5) AI 요약 / 리라이팅
- 문단 요약  
- 전체 기사 1문단 요약  
- SNS용 짧은 요약  
- SEO 강화 버전 자동 생성

---

### 🔵 6) AI 카테고리 추천 (kpopstar 구조 반영)
- 스타뉴스  
- 엔터테인먼트 → 음악 / 방송영화 / 공연

- 이슈

---

- 공격성·비하·성적 표현 감지  
- 논란 기사 필터 강화
